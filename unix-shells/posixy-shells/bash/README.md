# version/bash

A `bash` program that prints the running version of `bash`

# EXAMPLE

```console
$ ./lib/version
4.4.23(1)-release
```

# REQUIREMENTS

* [GNU bash](https://www.gnu.org/software/bash/)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
