# version/ash

An `ash` program that prints the running version of `ash`

# EXAMPLE

```console
$ ./lib/version
ii ash	0.5.7-4ubuntu1 compatibility package for dash
```

# REQUIREMENTS

* [ash](https://linux.die.net/man/1/ash)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
