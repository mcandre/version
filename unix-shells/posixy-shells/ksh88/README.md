# version/ksh88

A `ksh88` program that prints the running version of `ksh88`

# EXAMPLE

```console
$ ./lib/version
...
```

# REQUIREMENTS

* [ksh88](http://www.kornshell.com), sometimes packaged as "ksh".

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
