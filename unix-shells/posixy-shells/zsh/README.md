# version/zsh

A `zsh` program that prints the running version of `zsh`

# EXAMPLE

```console
$ ./lib/version
5.3
```

# REQUIREMENTS

* [zsh](http://www.zsh.org)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
