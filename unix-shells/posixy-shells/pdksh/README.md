# version/pdksh

A `pdksh` program that prints the running version of `pdksh`

# EXAMPLE

```console
$ ./lib/version
Version AJM 93u+ 2012-08-01
```

# REQUIREMENTS

* [pdksh](https://directory.fsf.org/wiki/Pdksh)

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast
```

## Test

```console
$ vast itest
```

## Lint

```console
$ vast lint
```
