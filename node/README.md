# version/node

A Node.js script that prints the server's version of Node.js

# EXAMPLE

```
$ npm pack
$ npm install -g version-0.0.0.tgz
$ version
{ http_parser: '1.0',
  node: '0.10.18',
  v8: '3.14.5.9',
  ares: '1.9.0-DEV',
  uv: '0.10.15',
  zlib: '1.2.3',
  modules: '11',
  openssl: '1.0.1e' }
```

# REQUIREMENTS

* [Node.js](http://nodejs.org/) 0.11.6+
