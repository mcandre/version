# version/java

A Java program that prints the version of the JRE running the program.

# EXAMPLE

```
$ mvn package
$ bin/version
1.7.0_40
```

# REQUIREMENTS

* [JDK](http://www.oracle.com/technetwork/java/javase/downloads/index.html) 1.7+

# CODE COVERAGE

```
$ mvn site
$ open target/site/coburtura/index.html
```
